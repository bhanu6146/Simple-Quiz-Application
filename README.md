# Simple-Quiz-Application :
Simple Quiz Application is a console-based Java program that allows users to answer multiple-choice questions and receive a score at the end of the quiz. The project demonstrates basic Java programming concepts such as conditional statements, user input handling, and score calculation.

Features :
- Interactive multiple-choice quiz
- Instant answer validation
- Automatic score calculation
- User-friendly console interface
- Beginner-friendly Java project
  
Technologies Used :
- Java
- Scanner Class
- Conditional Statements (if)
- Variables and Data Types
